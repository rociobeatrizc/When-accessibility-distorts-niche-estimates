# When accessibility distorts niche estimates: a virtual species framework across species prevalence and sample size gradients
## Scripts related to the paper 

Dear readers,

These are all the scripts related to the analysis made for our study. 
* [step1.R](https://github.com/rociobeatrizc/When-accessibility-distorts-niche-estimates/blob/main/step1.R): virtual species generation
* [step2.R](https://github.com/rociobeatrizc/When-accessibility-distorts-niche-estimates/blob/main/step2.R): an example of hypervolme calculation for a single species
* [step2_all_species.R](https://github.com/rociobeatrizc/When-accessibility-distorts-niche-estimates/blob/main/step2_all_species.R): same as step2, but for all the species
* [step3.R](https://github.com/rociobeatrizc/When-accessibility-distorts-niche-estimates/blob/main/step3.R): generation, for a single species, of their dissimilarity index (and area of applicability)
* [step3_all_species.R](https://github.com/rociobeatrizc/When-accessibility-distorts-niche-estimates/blob/main/step3_all_species.R): same as step3 but for all the species
* [DI_analysis.R](https://github.com/rociobeatrizc/When-accessibility-distorts-niche-estimates/blob/main/DI_analysis.R) takes all the Dissimilarity Index rasters generated in step3_all_species.R and does statistics and analysis. At the end, please find the relationship between species prevalence and number of biased occurrences section
* [hyp_analysis.R](https://github.com/rociobeatrizc/When-accessibility-distorts-niche-estimates/blob/main/hyp_analysis.R) takes all the hypervolumes of all the species and does statistics and analysis.
